# **📚 Study Session Logger (Python Tkinter Project)**

A simple desktop-based **Study Session Tracker** built using Python and Tkinter that helps students record, monitor, and analyze their daily study time subject-wise. The application supports both **manual entry** and **timer-based tracking**, and stores records locally for future reference.

---

## **Features**

* Timer-based study session tracking  
* Manual entry of study hours  
* Optional notes for each session  
* Date-wise study tracking  
* Subject-wise total study summary  
* Most studied subject detection  
* Session counter  
* Persistent storage using text files  
* Reset study history option  
* Auto-load last used subject

  ## **How to Run the Project**

  ### **Step 1: Install Python**

Download from:  
[https://www.python.org/downloads/](https://www.python.org/downloads/)

Check installation:

python \--version

### **Step 2: Run the Program**

Navigate to project folder and run:

python study\_tracker.py

The Study Tracker window will open automatically.

## **Example Workflow**

1. Enter subject name  
2. Add manual hours **OR**  
3. Start study timer  
4. Stop timer after session  
5. View updated analytics instantly  
   ---

   ## **Output Summary Includes**

* Subject-wise study hours  
* Total overall study time  
* Date-wise study breakdown  
* Most studied subject  
* Number of sessions recorded  
  ---

  ## **Data Storage**

Study records are saved locally in:

study\_record.txt

Last entered subject is saved in:

last\_subject.txt

This allows automatic restoration when reopening the application.
