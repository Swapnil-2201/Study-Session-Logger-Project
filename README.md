#Study Session Logger (Python Tkinter Project)
This project is a simple desktop-based Study Session Tracker built using Python and Tkinter. It helps users record daily study hours subject-wise using both manual input and a timer system. The application stores records locally so they remain available after restarting the program.
##Features
Add study hours manually
Track study sessions using a timer
Save subject-wise records automatically
Store notes for each study session
Track study hours date-wise
Show most studied subject
Display total number of study sessions
Automatically remember last entered subject
Reset all stored history if needed
Persistent storage using text files
##Technologies Used
Python
Tkinter (GUI)
File Handling
Threading
Datetime module
##Files Used
File Name
Purpose
study_record.txt
Stores study session history
last_subject.txt
Stores last entered subject

##How to Run the Project
Install Python (3.x recommended)
Save the program file as: study_tracker.py
Run the program:
python study_tracker.py
The application window will open automatically.
##Example Usage
Enter subject name
Add hours manually OR start timer
Stop timer after study session
View summary in the display panel
